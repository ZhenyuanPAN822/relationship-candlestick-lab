关系 K 线实验室

把聊天记录，
变成一张关系 K 线图

系统会逐条量化每一条消息，将暧昧、冷淡、试探、冲突与修复转化为关系指数， 再按 5 分钟、1 小时、日线、周线等周期聚合成 OHLC K 线。

你可以看到一段关系如何升温、回落、试探、修复， 以及每一次推进之后的承接力和抛压。

01

本地分析（Skills 模式）

使用本地 Claude Code / Codex 配合 Skills 完成分析，再将评分文件导入网页。

  • 在本地完成对每条消息的关系指数评分
  • 导出评分结果文件
  • 上传文件后直接生成关系 K 线图
  • 聊天内容从未上传任何外部服务
已有本地评分文件？导入后即可查看 K 线、成交量与指数路径。
02

在线分析（API 模式）

使用 API 在线分析聊天内容。

  • 上传聊天记录文件
  • 选择分析模型
  • 系统逐条识别暧昧、冷淡、试探、冲突与修复
  • 自动生成多周期关系 K 线
适合快速体验；如涉及敏感聊天内容，建议优先使用本地分析。

本地分析 · 导入评分文件

提交前你需要在本地完成评分（聊天记录不上传到任何外部服务）：

  1. 在你常用的 IDE（如 Claude Code、Codex）中调用配套 Skill
  2. 把聊天记录文件交给它（CSV / JSON / TXT 都可以）
  3. 等它跑完逐条量化 + 聚合（每 1000 条约 7 分钟）
  4. 它会输出一份本地评分文件并告诉你存放路径

把那个路径粘到下面的输入框，即可生成 K 线图。
